## 关键决策
- 保留现有 `cms/` React UI，避免重造后台界面。
- 使用 Cloudflare Pages Functions 提供 `/api/cms/*`。
- Functions 通过 GitHub API 读写仓库，不在浏览器暴露 GitHub Token。
- 第一版支持文章列表、读取、新建、保存、草稿/置顶切换；图片上传和站点配置编辑后续补。
